# Convertly

Convertly is a modern, dynamic desktop application for downloading and converting media from YouTube, Spotify, and more.

## Features

- **Dynamic UI**: Adapts to the platform (Spotify Green, YouTube Red).
- **Multi-Platform**: Supports YouTube, Spotify, SoundCloud, TikTok, etc.
- **Formats**: Convert to MP3 or MP4.
- **Modern Design**: Sleek Electron-based interface.

## Tech Stack

- **Frontend**: Electron, React, TailwindCSS
- **Backend**: Python (Flask), yt-dlp, spotipy, ffmpeg
